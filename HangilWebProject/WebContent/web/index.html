예제 페이지 입니다.

    이번 프로젝트는 웹표준을 지키지 않은 한길사 페이지를 홈페이지를 웹표준을 지킨 페이지로 리뉴얼 하는 것입니다.

     지금 보고 있는 이 페이지는 부트스트렙에 있는 탬플릿을 사용했습니다. 이 레이아웃을 기반으로 한길사에 어울리는 디자인으로 변경하는 작업을 하려합니다. 현 상태에서 변경하길 원하는 부분과 새로이 적용하고 싶은 부분과 관련된 의견은 언제든지 연락주시길 바랍니다.
     작업을 하실때는 web폴더에 새로운 html파일을 만들어서 index.html파일 내용을 전부 복사한 후 붙여 넣고 컨테이너 안에 표시한 영역 안에서 마음껏 페이지를 제작하시면 됩니다. jQuery파일은 javascriptFiles폴더에 위치하고 있으며 각자 생성한 javascript파일은 이 javascriptFiles폴더에 저장하면 됩니다.
     부트스트랩을 이용하여 반응성 웹이 되도록 클레스에 부트스트랩 기능을 지정해 주세요. 자신만의 스타일을 적용하고 싶을 때는 cssFiles폴더에 css파일을 만들고 html 20번째 줄 바로 아래에 파일 경로를 추가해주시면 됩니다. 단, 스타일을 적용 할 때는 컨테이너 내부의 스타일만 적용해주시길 바랍니다.
     저희 git 저장소의 주소는 이곳! 이고, 이클립스에 깃허브 연동을 못했으면 저장소 페이지 우측에 있는 Download ZIP을 누르면 저장소에 저장되어 있는 파일을 압축한 압축파일을 다운받을 수 있습니다. 파일업로드는 업로드 하려는 해당 폴더로 이동 후에 화면 중앙에 있는 UploadFile을 이용하여 업로드 하면 되고, 수정, 삭제하려는 파일은 대상 파일을 선택 후에 수정, 삭제를 하면 됩니다. 다들 마지막 프로젝트 즐겁게 진행합시다! 화이팅!
